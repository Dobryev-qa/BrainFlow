# BrainFlow — Текст для Chrome Web Store (RU)

## Короткое описание
Плавающий фокус-виджет для концентрации: таймер, анти-прокрастинация, статистика и календарь.

## Полное описание
BrainFlow помогает держать фокус во время работы и учебы в интернете.

Что внутри:
- Плавающий виджет на любой странице
- Таймер фокуса в режимах 10/3 и 15/5
- Старт, пауза, сброс и автопереход между блоками
- Кнопка «Задача выполнена» и визуальный прогресс дня
- Мягкий перехватчик прокрастинации при смене домена и бездействии
- Статистика по дням и календарь месяца с деталями
- Поддержка Русского и English
- 4 визуальные темы для комфортной работы

Конфиденциальность:
- Все данные хранятся локально в браузере
- Нет аккаунтов, внешних серверов и трекинга

Для кого:
- Люди с СДВГ
- Студенты
- Удаленные специалисты
