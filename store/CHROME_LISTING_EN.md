# BrainFlow — Chrome Web Store Listing (EN)

## Short description
Floating focus widget with timer, anti-procrastination nudges, and monthly calendar stats.

## Full description
BrainFlow helps you stay focused while working or studying online.

Features:
- Floating widget on every page
- Focus timer with 10/3 and 15/5 modes
- Start, pause, reset, and automatic work/break flow
- “Task done” button and daily visual progress
- Gentle anti-procrastination prompts on domain change and inactivity
- Daily stats + monthly calendar with clickable day details
- Russian and English support
- 4 visual themes for comfortable long sessions

Privacy:
- All data is stored locally in browser storage
- No accounts, no external servers, no tracking

Built for:
- People with ADHD
- Students
- Remote professionals
