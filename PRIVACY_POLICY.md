# BrainFlow Privacy Policy

Last updated: February 22, 2026

BrainFlow stores all user data locally in the browser using extension storage.

## What we collect
- Timer state (running/paused, remaining time, selected mode)
- Widget position
- Daily productivity metrics (cycles, brains, focus minutes)
- Local settings (language, default mode, sound/interceptor toggles)

## What we do not collect
- No account data
- No personal identifiers
- No browsing history uploads
- No external analytics
- No third-party tracking scripts

## Data sharing
BrainFlow does not transmit user data to any external server.

## Data control
Users can reset all productivity statistics from the settings page.

## Contact
Publisher contact can be added before store submission.
